# Samba - Dateifreigabe

Samba ermöglicht dir den Zugriff auf die Dateien des Servers über das Netzwerk. Du kannst von deinen Geräten auf deine Dateien zugreifen, als wären sie lokal gespeichert.

## Verbindung herstellen

### macOS

1. Öffne den **Finder**
2. Drücke **⌘ + K** (oder "Gehe zu" → "Mit Server verbinden")
3. Gib ein: `smb://beelink`
4. Klicke auf **Verbinden**
5. Gib deine Anmeldedaten ein
6. Wähle die gewünschte Freigabe aus

**Automatisch verbinden:**
- Nach erfolgreicher Verbindung erscheint der Server in der Finder-Seitenleiste
- Du kannst ihn zu den Anmeldeobjekten hinzufügen (Systemeinstellungen → Benutzer & Gruppen → Anmeldeobjekte)

### iOS

1. Öffne die **Dateien-App**
2. Tippe auf die **drei Punkte** (···) oben rechts
3. Wähle **Mit Server verbinden**
4. Gib ein: `smb://beelink`
5. Tippe auf **Verbinden**
6. Gib deine Anmeldedaten ein
7. Der Server erscheint unter "Geteilt" in der Seitenleiste
